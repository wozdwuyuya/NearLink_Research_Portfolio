# NearLink Research Portfolio

本仓库集合了基于星闪(NearLink)通信协议的医疗物联网(IoMT)系列研究实现。

## 项目结构
- **01_Routing_Algorithm_Simulation**: 基于模糊免疫算法的无线传感网路由调度优化。
- **02_Cross_Layer_Optimization**: 针对Hi3863驱动的可靠性审计与跨层优化工程方案。
- **03_ML_Cognitive_Diagnosis**: 基于BiLSTM与RAG的通信链路质量在线诊断与认知推理系统。

## 技术栈 (Technology Stack)
- **Languages**: C, Python, LaTeX
- **Core Technologies**: NearLink (星闪), PyTorch, FAISS
- **AI Methodology**: Multi-Agent Static Audit, RAG (Retrieval-Augmented Generation), Uncertainty Quantification (MC Dropout)

## 如何开始
请参考各子文件夹下的源代码注释与设计文档。对于需要环境配置的项目，请参考该目录下的 `requirements.txt` 或环境说明。
